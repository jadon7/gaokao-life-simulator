# 角色 / 冲突模板库

这份文档解决的是：

**18 张牌里，到底该让谁来制造冲突，谁来承接关系线，谁适合做回收点。**

如果只有出题结构，没有角色模板，剧情很容易变成：

- 每年都像匿名 NPC 在发题
- 每次冲突都成立，但不记人
- 关系线像凭空出现

这份文档的目标，就是给 18 张牌准备一套稳定的“人物发动机”。

---

## 一、角色库的职责

每个角色至少承担下面四种功能中的一种：

1. 推剧情
2. 逼选择
3. 承接关系线
4. 回收前文

角色不要太多。

对 18 张牌来说，**长期活跃角色建议控制在 5-7 个**。

再多就会散，再少就会显得世界太空。

---

## 二、推荐的角色骨架

建议固定成这 7 类：

1. 室友 / 同伴型
2. 学长 / 前辈型
3. 导师 / 老师型
4. 家庭型角色
5. 关系线核心角色
6. 外部机会角色
7. 团队 / 同学 / 同事群像

其中真正需要反复回收的核心人物，通常是：

- 室友 / 同伴
- 关系线核心角色
- 1 个前辈或导师
- 1 个家庭角色

---

## 三、核心角色模板

## 1. 室友 / 同伴型

### 角色职责

- 最适合把玩家“推上场”
- 最适合制造早期生活线冲突
- 最适合做轻喜剧、社死、误打误撞的推进器

### 适用幕

- 第一幕最强
- 第二幕和第三幕可以继续回收
- 后面更适合转成“你现在和他已经不是原来那个站位了”

### 常见冲突

- 先把事搞起来，再逼你收拾后果
- 过度乐观、先答应、先报名、先拉群
- 好心办坏事
- 他替你制造机会，也替你制造麻烦

### 适合绑定的 A/B 类型

- `先动手补 / 先分析清`
- `抢机会 / 守底盘`
- `先独自扛 / 先拉人一起`

### 适合的 callback

- 第一次拉你进项目
- 第一次翻车事故
- 第一次给你起外号
- 后面他是否还站在你这边

### 示例定位

- 周越这种人最适合做：
  - 开局发动机
  - 气氛制造者
  - 中段的现实对照组

---

## 2. 学长 / 前辈型

### 角色职责

- 把“校内生活”推向“外部机会”
- 让主角第一次觉得自己被更大世界看见
- 也负责把代价升级

### 适用幕

- 第二幕最强
- 第三幕和第五幕适合回收

### 常见冲突

- 给你一个看起来很值的机会
- 让你在课业 / 稳定 / 现实之间做更难的取舍
- 让你第一次尝到外部评价的甜头

### 适合绑定的 A/B 类型

- `抢机会 / 守底盘`
- `先赌增长 / 先控风险`
- `先保体面 / 先保结果`

### 适合的 callback

- 第一次被夸
- 第一次路演
- 第一次外部项目
- 后面再回来问你要不要更大一步

### 注意

- 前辈不能总是正面贵人
- 最好既带来机会，也带来压力税

---

## 3. 导师 / 老师型

### 角色职责

- 负责现实秩序
- 负责长期评价体系
- 让玩家感受到“不是所有东西都靠热血冲掉”

### 适用幕

- 第一幕埋下“被看见”
- 第三幕、第四幕最适合做压力来源
- 第六幕适合做某种迟来的认可

### 常见冲突

- 临时加码
- 公开评价
- 把难活递给你
- 对你有期待，但不负责替你兜底

### 适合绑定的 A/B 类型

- `先保体面 / 先保结果`
- `先公开表达 / 先私下处理`
- `先动手补 / 先分析清`

### 适合的 callback

- “老师默认你能兜底”
- “他第一次认真记住你”
- “之前给你的那句话，后面回头成真了”

### 注意

- 老师型角色不要写成纯反派
- 更像秩序和长期后果的代言人

---

## 4. 家庭型角色

### 角色职责

- 提供现实世界的重量
- 让选择不只是校园里的游戏
- 给“风险承受力”“期待”“情绪压力”落地

### 适用幕

- 第一幕可轻触
- 第四幕以后要明显变重
- 第五、第六幕适合做价值回响

### 常见冲突

- 对稳定的期待
- 对专业、城市、收入、关系的看法
- 一通电话、一句比较、一个现实难题

### 适合绑定的 A/B 类型

- `先做自己 / 先顾期待`
- `先赌增长 / 先控风险`
- `先保体面 / 先保结果`

### 适合的 callback

- 家里最开始说过的话
- 你答应过但没做到的事
- 某次真正开始理解彼此

### 注意

- 家庭线不要只会催
- 最好既有压力，也有某种笨拙支持

---

## 5. 关系线核心角色

### 角色职责

- 承接整局的关系副线
- 提供“不是所有代价都能补票”的那层情感现实
- 让便签真正有暗流

### 适用幕

- 第一幕种子
- 第二幕露头
- 第三幕拉扯
- 第四、第五幕翻面
- 第六幕给结果

### 常见冲突

- 不是直接恋爱，而是：
  - 多看一眼
  - 会记得你说过的话
  - 会主动留位置
  - 会因为你没回应而往后退

### 适合绑定的 A/B 类型

- `先回应人 / 先完成事`
- `先公开表达 / 先私下处理`
- `先保体面 / 先保结果`

### 适合的 callback

- 一张合照
- 一次没吃成的饭
- 一条撤回过的消息
- 一个你随口提过却被对方记住的细节

### 注意

- 关系线核心角色不能只在主问题里出现
- 更重要的是在便签里持续推进

---

## 6. 外部机会角色

### 角色职责

- 代表社会化入口
- 负责把“校园任务”变成“真实世界任务”

### 适用幕

- 第二幕、第五幕最强
- 第六幕可回收成最终机会或最终试炼

### 常见冲突

- 合作方
- 甲方
- 实习主管
- 创业园 / 平台 / 组织方

### 适合绑定的 A/B 类型

- `抢机会 / 守底盘`
- `先赌增长 / 先控风险`
- `先保体面 / 先保结果`

### 适合的 callback

- 第一次外部认可
- 第一次真正被要求交付
- 最后一次你还要不要继续往前走

### 注意

- 外部机会角色不需要写得很饱满
- 更重要的是功能性清楚：它负责放大你的现实代价

---

## 7. 团队 / 同学 / 同事群像

### 角色职责

- 提供环境氛围
- 提供比较、流言、舆论、局部压力
- 让主角不是活在真空里

### 适用幕

- 全程都可以有
- 但不适合做唯一核心冲突

### 常见冲突

- 大家怎么看你
- 某次公开场合的回声
- 你在小圈子里的位置变化

### 适合绑定的 A/B 类型

- `先公开表达 / 先私下处理`
- `先保体面 / 先保结果`
- `先独自扛 / 先拉人一起`

### 适合的 callback

- 某个外号
- 某次公开社死
- 某次被截图转发

---

## 四、角色在六幕里的推荐分布

## 第一幕

- 室友 / 同伴：主推手
- 导师 / 老师：秩序背景
- 关系线核心角色：只种子，不摊牌

## 第二幕

- 学长 / 前辈：主推机会
- 室友 / 同伴：加速器
- 关系线核心角色：开始有存在感

## 第三幕

- 导师 / 老师：加重代价
- 关系线核心角色：开始拉扯
- 团队群像：放大压力

## 第四幕

- 家庭型角色：明显变重
- 关系线核心角色：第一次翻面
- 外部机会角色：逼你做现实选择

## 第五幕

- 外部机会角色：升级难度
- 家庭型角色：现实重量落地
- 室友 / 同伴 / 前辈：作为旧线回收

## 第六幕

- 关系线核心角色：收口
- 家庭型角色：和解或确认
- 导师 / 前辈 / 外部机会角色：做最终回响

---

## 五、角色和便签的绑定原则

便签最适合承载的，不是新剧情，而是：

- 某个人的态度变化
- 某段关系的温度变化
- 某个长期后果的累积

### 适合放进便签的人

- 关系线核心角色
- 家庭型角色
- 导师 / 老师
- 室友 / 同伴

### 不太适合大量放进便签的人

- 外部机会角色
- 群像角色

因为这两类更适合做事件推动，不适合做细腻温度推进

---

## 六、角色复用检查表

如果一个角色满足下面 3 条以上，就值得回收：

- 第一次出场时留下了明确记忆点
- 他的存在改变过主角的路线
- 他代表一种长期矛盾
- 他能在后面带来情绪回响
- 他能和某个 callback 绑定

如果不满足，后面就别硬回收。

---

## 七、最推荐反复回收的 6 类 callback

1. 一次没吃成的饭
2. 一条撤回过的消息
3. 第一次公开被夸
4. 第一次翻车事故
5. 家里说过的一句话
6. 某个外号 / 某次截图 / 某个被记住的小细节

这些都非常适合：

- 前面轻埋
- 中段变重
- 后面回响
